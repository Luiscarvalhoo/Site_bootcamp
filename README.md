# 🌐 Meu Portfólio - Bootcamp 2025

Este site foi criado utilizando a plataforma **Google Sites** como parte da **entrega inicial do bootcamp** com o professor Marcelo Carboni Gomes em 2025.

🔗 Acesse o site aqui: [https://sites.google.com/view/portfolioohub/home](https://sites.google.com/view/portfolioohub/home)

## 📄 Sobre

A proposta era criar um portfólio online que apresentasse:

- Informações pessoais
- Contatos e redes sociais
- Citação motivacional
- Organização visual simples e objetiva

Como a plataforma **Google Sites** não permite acesso direto ao código-fonte completo, este repositório serve apenas como referência e documentação da entrega.
